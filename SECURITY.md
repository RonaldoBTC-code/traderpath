# Seguridad

TraderPath opera exclusivamente con capital virtual. Nunca deben añadirse claves privadas, credenciales de exchange, datos de pago ni secretos de Supabase al repositorio.

## Reportar una vulnerabilidad

Utiliza la opción de reporte privado de vulnerabilidades del repositorio de GitHub. No publiques detalles explotables en un issue abierto.

Incluye:

- componente y versión afectados;
- pasos mínimos para reproducir;
- impacto esperado;
- mitigación propuesta, si existe.

## Dependencias

Dependabot revisa dependencias npm semanalmente y GitHub Actions mensualmente. Los cambios de versión mayor requieren validación manual del flujo educativo, autenticación, RLS y persistencia.
